## Our dataset
First download our dataset from https://anonfiles.com/8cx3ffI7y9/ours_zip. Then, unzip it to the './data' fold as follows:
```
|-- code
    |-- data      
        |-- Matterport3D
        |-- Stanford2D3D
        |-- ours 
            |-- bedroom  
            ...
```
Finally, run the command
```
sh ours.sh
```
